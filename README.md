## 学生信息管理系统-窗体应用程序简单说明


### 项目简介

- C#课程期中项目
- 简单的窗体应用程序
- 简单的学生信息管理系统
- 拥有学生端、教师端、后台管理

### 功能特征

- 采用Access数据库,数据存放简单
- C#编程

### 环境依赖

- `OS Windows 7 x64`
- `开发环境：Visual Studio 2010旗舰版`
- `数据库：Access 2010`

### 项目说明
- 项目仅供学习参考，已停止更新
